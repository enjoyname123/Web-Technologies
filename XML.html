What is XML?

XML (eXtensible Markup Language) is used to store and transport data. It uses user-defined tags and follows strict syntax rules.

1. XML Declaration

<?xml version="1.0" encoding="UTF-8"?>

2. Complete XML Document

<?xml version="1.0" encoding="UTF-8"?>

<!-- Processing Instruction -->
<?xml-stylesheet type="text/xsl" href="style.xsl"?>

<!-- DTD Definition -->
<!DOCTYPE college [
    <!ELEMENT college (student+)>
    <!ELEMENT student (name, course, age, marks, address, notes)>
    <!ELEMENT name (#PCDATA)>
    <!ELEMENT course (#PCDATA)>
    <!ELEMENT age (#PCDATA)>
    <!ELEMENT marks (#PCDATA)>
    <!ELEMENT address (#PCDATA)>
    <!ELEMENT notes (#PCDATA)>
    <!ATTLIST student id ID #REQUIRED>
]>

<college xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">

    <student id="S101" status="active">
        <name>Sathya</name>
        <course>AI &amp; ML</course>
        <age>20</age>
        <marks>85</marks>
        <address>Hyderabad &lt;Telangana&gt;</address>

        <notes><![CDATA[
            XML CDATA section
            allows special characters &lt; &amp; &gt;
        ]]></notes>
    </student>

</college>

3. XML Features Demonstrated

  • XML Declaration
  • Root Element
  • Nested Elements
  • Attributes
  • Comments
  • Character Data
  • Special Characters (Entities)
  • CDATA Section
  • DTD (Internal)
  • Processing Instruction
  • Namespace

4. XML vs HTML

XML HTML
Stores data Displays data
User-defined tags Predefined tags
Case-sensitive Not case-sensitive
Strict syntax Flexible syntax

5. Viva Notes

DTD: Defines the structure and rules of an XML document.
CDATA: Used to include text that should not be parsed.
Entity: Used to represent special characters.
Namespace: Avoids name conflicts in XML.
